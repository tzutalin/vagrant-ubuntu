# vagrant-ubuntu

Use Vagrant to init virtualbox with Ubuntu16.04

## Requirements
### Necessary
* Vagrant > 1.8.1
* Virtualbox

#### Optional
* Ansible
* Terraoform

## Usage

##### Start
```
cd vagrant-ubuntu
vagrant up
```
You can try to $ ping 192.168.50.4 from your host 

##### SSH
```
vagrant ssh
```

#### Destroy
```
vagrant destroy
```
